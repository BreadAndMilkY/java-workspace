# MCP

- [MCP Server 工具与插件](https://mcp.so/zh)

### 一、MCP 简介

MCP（Model Context Protocol）是一种模型上下文协议，主要用于AI模型调用外部工具和函数。

### 二、MCP 传输方式

#### 1. STDIO（标准输入输出）

- 通过标准输入输出流进行数据传输
- 适用于命令行工具和本地进程间通信
- 数据以文本流形式传输，简单高效
- 常用于开发调试和本地测试环境

基于 stdio 的 MCP 服务端通过标准输入输出流与客户端通信，适用于作为子进程被客户端启动和管理的场景，非常适合嵌入式应用。

#### 2. SSE（Server-Sent Events）

- 基于 HTTP 的单向服务器推送技术
- 服务器可以主动向客户端发送上下文更新
- 支持实时数据流传输
- 适用于需要服务器主动推送上下文变更的场景

基于SSE的MCP服务端通过HTTP协议与客户端通信，适用于作为独立服务部署的场景，可以被多个客户端远程调用。

#### 3. Streamable HTTP

- 基于 HTTP 协议的流式传输
- 支持分块传输和流式响应
- 可以处理大体积的上下文数据
- 适用于需要传输大量上下文信息的场景
- 支持断点续传和流控制
